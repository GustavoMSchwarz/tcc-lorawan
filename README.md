# Monitoramento Agrícola com LoRaWAN - Campo Alegre/SC

Este repositório contém o desenvolvimento do meu **TCC II** para o curso de **Bacharelado em Sistemas de Informação (UDESC/CEPLAN)**.

## 🎯 Objetivo do Projeto
Implementar e avaliar uma rede de comunicação sem fio utilizando a tecnologia **LoRaWAN** e um **Gateway Cisco** para o monitoramento de variáveis agrícolas na região de Campo Alegre - SC.

## 🛠️ Tecnologias Utilizadas
* **Hardware:** Gateway Cisco (LoRaWAN), Sensores (ESP32/Arduino).
* **Software:** Python (Tratamento de dados), QGIS (Mapeamento de cobertura).
* **Infraestrutura:** Project IDX & GitHub.
